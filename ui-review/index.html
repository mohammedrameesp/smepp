How to Use This

  1. 1 Open any HTML page in your browser (or use Live Server in VS Code)
  2. 2 Edit the HTML/CSS directly to improve the UI design
  3. 3 Once satisfied, apply the changes to the corresponding React components in src/app/
  4. 4 Use browser DevTools (F12) to experiment with styles in real-time
